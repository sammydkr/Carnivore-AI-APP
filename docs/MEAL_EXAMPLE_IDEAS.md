These are future improvements for the meal example and quick-fill system. The first version only includes four simple quick-fill buttons, but this list captures where the feature can go next.

- Ketovore meal
- Budget meal

## Score Preview

Show an expected result before the user analyzes:

- Expected score: `10/10`
- Low carb
- High protein
- Whole-food focused

## Food Education Notes

When the user taps an example, show a short explanation:

```text
Steak + eggs is high-protein, low-carb, and carnivore-friendly.
```

## Meal Size Options

Let users scale an example:

- Small
- Medium
- Large
- OMAD

The app can adjust grams and egg count automatically.

## Warning Meal Examples

Add example meals that intentionally show lower scores so users learn what to avoid:

- Steak + fries
- Chicken + rice
- Eggs + toast
- Salmon + seed oil sauce

## Learn By Example Cards

Create an educational comparison section:

```text
Good Choice
Steak + eggs
Score: 10/10

Needs Review
Chicken + rice + seed oil sauce
Score: 4/10
```
